Суть проекта:
Сайт на Flask с использованием изученных технологий
(Библиотеки flask_wtf, flask, flask_login, dotenv, datetime, os, smtplib, mimetypes, email.encoders, email.mime.audio, email.mime.base, email.mime.multipart, email.mime.text, email.mime.image)
Сайт представляет собой task-manager, с авторизацией и регистрацией, заметки создаются на сайте, также есть возможность продублировать информацию на почту пользователю. Заметки можно изменять и удалять.
